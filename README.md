<h1>change_mac.py</h1>
<p>Python script, works with all versions currently.</p>
<p>Create and tested in Linux, no compatibility Windows or MacOS currently.</p>
<p>Basic help is available if you run the script with --help</p>
<h4>$ python mac_changer.py --help</h4>
<p>To run the script please provide arguments in the following format. 
From the command line inside the "Change_Mac" Project Folder.</p>
<h3>
python[#] change_mac.py -i [INTERFACE] -m [XX:XX:XX:XX:XX:XX]
</h3>
<h4>Press enter and three things should happen</h4>
<ol>
<li>Confirmation of the interface and new MAC you requested.</li>
<li>ifconfig will execute for your interface, to provide more details.</li>
<li>You will get confirmation of the update.  Or an error message.</li>
</ol>
