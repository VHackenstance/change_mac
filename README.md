# change_mac.py

### Python script, works with all versions currently.
### Create and tested in Linux, no compatibility Windows or MacOS currently.
### To run the script please provide arguments in the following format.
### From the command line inside the "Change_Mac" Project Folder.

## python [Your Version] change_mac.py -i (or --interface) [INTERFACE] -m (or --mac) [NEW_MAC_ADDRESS]
## Press enter and three things should happen
### 1. Confirmation of the interface and new MAC you requested.
### 2. ifconfig will execute for your interface.
### 3. A test will run in the background and you will get confirmation of update.
